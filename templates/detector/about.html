About The Project

How This Real-Time Detection System Works

This student-friendly project combines Django for the web application, OpenCV for image and video processing, and Ultralytics YOLO for object detection.

Browser Webcam Frame API Method

  • Browser asks permission for webcam access using JavaScript
  • Frames are captured from a hidden video element using a canvas
  • Frames are sent to Django using fetch and JSON
  • Django runs YOLO inference and returns an annotated frame
  • The page updates the preview continuously for live detection

OpenCV Plus Django MJPEG Method

  • Django opens the webcam directly with OpenCV
  • Each frame is annotated on the backend
  • Frames are streamed as MJPEG to an image tag on the page
  • This approach is easier to explain and often more stable for a student project

Recommendation

For a polished web experience, use the browser webcam method. For easy demonstration and fewer frontend issues, the OpenCV MJPEG streaming method is usually the easiest and most stable student-project option.
